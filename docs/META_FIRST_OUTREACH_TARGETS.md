# Meta / Llama First Outreach Targets

## Current Recommendation

The first Meta / Llama outreach should run through three parallel paths:

1. Llama Startup Program or Llama ecosystem support path.
2. Meta Open Source AI or Llama Impact Grants path.
3. Direct human outreach to Llama developer ecosystem or open-source AI partnership people.

The first two paths are historically verified, but their current application status must be checked before sending. Do not assume an old program page is an active application.

## Path 1 — Llama Startup Program / Ecosystem Support

### Fit

JudgmentOps is early open-source AI workflow infrastructure. A Llama-oriented phase would create public workflow examples for creators, developers, and small teams, using protocol-governed agentic workflows rather than one-off prompts.

The historical Llama Startup Program offered hands-on technical support and reimbursement for hosted Llama API usage. That model of support could fund repeated execution and review runs for public paired evaluations.

### Verified Context

- Meta published the [Llama Startup Program](https://ai.meta.com/blog/llama-startup-program/).
- The initial cohort was for eligible early-stage US startups.
- The initial application deadline was **May 30, 2025 at 6:00 pm PT**.
- The initial benefits included potential reimbursement of up to **$6,000 USD per month for up to six months** and technical support.

This verifies the historical program, not a currently open 2026 cohort.

### Action

- Check the official program page for a new cohort, application, waitlist, or ecosystem contact.
- If a current application is open, confirm incorporation, geography, funding, developer, and product eligibility before applying.
- Prepare the submission using `docs/OUTBOUND_EMAIL_META.md` and `docs/META_SUBMISSION_BRIEF.md`.
- If no current application is open, use the page as evidence of ecosystem fit and seek the current Llama developer-program route.

**Status:** manual verification required before sending.

## Path 2 — Meta Open Source AI / Llama Impact Grants

### Fit

JudgmentOps may fit the broader themes of open-source AI adoption, economic progress, developer productivity, and expanding the capabilities of small teams.

The strongest framing is public infrastructure and reproducible evidence: a workflow protocol, Llama-oriented fixtures, and published comparisons of ordinary prompts with governed execution loops.

### Verified Context

- The [Meta Open Source AI page](https://ai.meta.com/opensourceai/) describes the Llama Impact program as including grants, awards, hackathons, and community-building programs.
- Meta's [Llama 3.1 Impact Grants announcement](https://about.fb.com/news/2024/08/encouraging-impactful-uses-of-ai-llama-impact-grants/) described a historical global open call with up to $2 million in total awards.
- Meta later announced the [second-round recipients](https://about.fb.com/news/2025/04/llama-impact-grant-recipients/) and described accelerator programs, hackathons, competitions, workshops, and technical mentorship.
- Meta has also run geography-specific programs, including a [Sub-Saharan Africa Llama Impact Accelerator](https://about.fb.com/news/2025/06/accelerating-open-source-innovation-across-sub-saharan-africa-introducing-the-llama-impact-accelerator-program/).

These sources establish historical program activity. They do not establish a currently open general grant.

### Action

- Check current Meta Open Source AI grant and award pages.
- Confirm whether any current opportunity is global, regional, academic, nonprofit, or startup-specific.
- Do not apply to a region-specific grant unless eligible.
- If no active general grant exists, use this path for the project narrative and ecosystem routing rather than a formal application.

**Status:** manual verification required before sending.

### Verification Before Adding A Person

- Confirm the person currently works at Meta.
- Confirm the role is relevant to Llama, open-source AI, developer ecosystem, partnerships, or agent workflows.
- Use only a public professional profile or an official contact route.
- Do not infer or generate an email address.
- Prefer asking for routing help over requesting funding in the first message.

### Outreach Style

- Short note.
- Ask for the correct reviewer or contact.
- Do not hard sell.
- Include repository, release, and reviewer brief.
- State that JudgmentOps is an early prototype and request feedback.
