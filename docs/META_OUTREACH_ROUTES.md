Prepare the first outreach routes for the JudgmentOps Meta / Llama package. Do not send yet.

## Primary Message

Use `docs/OUTBOUND_EMAIL_META.md` as the canonical email.

**Status:** manual research required

**Reason:** Most relevant because JudgmentOps is positioned as an open-source judgment layer for Llama-powered agentic workflows.

**What to look for:**

- Llama official contact or ecosystem form.
- Llama open-source community contact.
- Meta AI / Llama developer relations contact.
- Llama grant, impact, or startup program application page.
- Llama GitHub, Hugging Face, or community discussion route if appropriate.

## Route B — Meta AI / Llama People On LinkedIn Or X

**Status:** manual research required

**Status:** manual research required

## Route E — Warm Introduction Route

**Status:** manual research required

**Potential sources:**

- Existing AI, game, or open-source contacts.

- Investors or ecosystem operators.
- AI founders using Llama.

**Ask:**

> Who is the right Meta / Llama ecosystem person to review an open-source agent workflow governance project?

## First Send Recommendation

Priority order:

1. Official Llama or open-source AI program if currently open.
2. Direct outreach to 3–5 relevant Meta / Llama ecosystem people.
3. Warm introduction request if a reliable path exists.
4. GitHub or community route only if non-spammy and clearly relevant.

## Do Not Use

- Generic Meta support forms for consumer products.
- Facebook business support.
- Instagram or WhatsApp support.
- Random issue spam on unrelated repositories.
- Claims of affiliation, endorsement, or Meta support.
